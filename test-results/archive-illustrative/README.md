# Archived — illustrative planning documents

The two files in this folder (`dr-test-2024-10-01.md`, `dr-test-2024-11-15.md`) were written as **planning / template documents** describing what a DR test report would look like. They were not the output of a real executed test.

They are preserved here because the format, metrics, and structure are still useful as a template for writing up real DR test reports.

Real DR test results from this point forward live in `test-results/` at the repo root, one file per test date, produced from actual failover exercises against the Hyper-V lab pair.
